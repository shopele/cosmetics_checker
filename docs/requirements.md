# 要件仕様書

**アプリ名**: 化粧品・医薬部外品 表記チェッカー  
**作成日**: 2026-05-19  
**最終更新**: 2026-05-20  
**バージョン**: 1.2

---

## 1. 目的

製品パッケージ画像を複数枚アップロードし、薬機法の必須記載事項を Claude API（Vision）で自動チェックする業務ツール。

---

## 2. 機能要件

| # | 機能 | 説明 |
|---|---|---|
| F01 | 画像アップロード | 複数枚の製品パッケージ画像をアップロード（ドラッグ&ドロップ対応） |
| F02 | カテゴリ選択 | 化粧品 / 医薬部外品 の2種類から選択 |

| F05 | 結果表示 | 全体判定バナー＋項目別判定結果テーブルを表示 |
| F06 | 履歴保存 | チェック結果をlocalStorageに保存（最新50件） |
| F07 | 履歴表示 | 過去のチェック結果一覧を表示、詳細表示も可 |
| F08 | CSVエクスポート | 履歴データをCSVファイルとしてダウンロード |
| F09 | 免責表示 | AIによる参考判定である旨を常時表示 |

| F13 | NG表現チェック | 効能効果標榜などの NG 表現を検出して結果表示 |
| F14 | 抽出テキスト表示 | AI が画像から読み取ったテキスト全文を折りたたみ表示 |
| F15 | カスタム項目 | 社内基準のチェック項目を追加・削除して判定対象に含める |
| F16 | 結果メモ・履歴フィルター | 結果にメモ保存、履歴のカテゴリ/判定/日付/キーワード絞り込み |

## 4. チェック設計方針

**Approach B**: AIは事実判定のみ

- チェックルールを `js/rules.js` にJSONで明示的に定義
- Claude APIの役割は「画像内に記載があるか否かの事実判定のみ」
- 適否判断・法的解釈はAIに求めない
- 各項目に根拠条文（薬機法の条番号）を紐付け

## 6. 判定仕様

| 判定値 | 表示 | 意味 |
|---|---|---|
| found | ✓ 記載あり | 画像内に記載が確認できた |
| not_found | ✗ 記載なし | 画像内に記載が確認できなかった |
| unclear | ? 判定不可 | 画像が不鮮明等で判定できなかった |

**全体判定**:
- 全必須項目が `found` → 問題なし（緑）
- 1件でも必須項目が `not_found` → 要確認（赤）
- `not_found` なしで `unclear` あり → 要確認（黄）
